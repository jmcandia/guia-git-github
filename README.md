# 📘 Guía Git & GitHub

Guía interactiva de Git y GitHub diseñada para estudiantes de programación. Cubre desde los conceptos básicos hasta estrategias de branching usadas en la industria.

🔗 **[Ver guía en línea](https://jmcandia.github.io/guia-git-github)**

---

## Contenido

- Qué es Git y para qué sirve
- Instalación y configuración inicial
- Conceptos base: zonas de Git, commits, ramas
- Comandos esenciales del día a día
- Trabajando con GitHub y Pull Requests
- [Conventional Commits](https://www.conventionalcommits.org/es/v1.0.0/) — estándar para mensajes de commit
- [Gitmoji](https://gitmoji.dev) — emojis con significado en tus commits
- Estrategias de branching: Git Flow, GitHub Flow y Trunk-Based Development
- Cheatsheet de referencia rápida
